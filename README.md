# 1: Movie Recommendation System (Week 1)
# Simple Movie Recommendation System

# Objective -

To perform data handling and exploratory data analysis on a movie dataset and build a basic recommendation system using similarity measures.

# Description -

This project uses a movie dataset (IMDb/TMDb) to analyze textual features and recommend movies based on content similarity or popularity. The focus in Week 1 is on data preprocessing, feature extraction, and understanding similarity-based recommendations.

# Key Tasks (Week 1)

Dataset loading and inspection
Handling missing and inconsistent data
Text preprocessing for movie descriptions
Exploratory analysis of genres, ratings, and popularity

# Techniques Used

TF-IDF Vectorization
Cosine Similarity
Basic content-based filtering

# Tech Stack

Python
Pandas
Scikit-learn
Jupyter Notebook
